# Munamii Cakery — React Website (Front End Project 4)

A fully responsive **React** website for **Munamii Cakery**, featuring cupcakes, wedding cakes, a dynamic cart, admin tools, and a simulated checkout flow. It also features a email-subscription form and a contact form, both with validation. 
The project evolved from a simple static concept into a modern React application with reusable components, contexts, routing, and a polished bakery‑inspired UI.

---

## 🌐 Live Demo

**https://johan-fullstack-ai.github.io/react-cakery**

---

## 📁 Project Structure

- **Components** → UI building blocks  
- **Pages** → route-level screens  
- **Contexts** → global state (auth, cart, products)  
- **Layouts** → shared page chrome  
- **Routes** → navigation logic  
- **Assets** → images

This structure supports clean separation of concerns.

---

## 🧁 Details

### User-facing
- Mobile‑first responsive design  
- Cupcake & wedding cake product pages  
- Dynamic cart with:
  - Add/remove items  
  - Quantity controls  
  - LocalStorage persistence  
  - Total calculation  
- Simulated checkout popup (“Order placed!”)  
- Sticky header with navigation
- Subscribe form 
- Contact form

### Admin
- Admin login  (through the top-left logo)
- Admin dashboard  
- Admin product management page

## 🎨 Design & Branding

The design follows a soft, bakery‑inspired aesthetic:

- Pastel color palette  
- Rounded product images (16px radius)  
- Google Fonts: **Baloo 2**, **Pacifico**, **Nunito**  
- Turquoise gradient buttons  
- Subtle pastel dot background  
- Clean spacing and visual hierarchy
